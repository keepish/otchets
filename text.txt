5.4 Составление тестового сценария (тест-кейса)
Оформить набор тестов из п.5.3 в виде тестового сценария. Для каждого указать:
- № тест-кейса,
- приоритет теста (низкий, средний, высокий),
- название теста,
- шаги тестирования (все действия для воспроизведения теста),
- данные тестирования (какие данные должны быть введены пользователем),
- ожидаемый результат,
- фактический результат,
- предпосылки (что сделать до тестирования и ввода данных пользователем,
например: что должно быть добавлено в БД до тестирования),
- постусловия (что сделать после тестирования, например: удалить определенные
данные из БД, созданные файлы),
- статус (Pass/Fail),
- комментарии.
